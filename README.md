
#### 《坦白书》 马良

```
我所有的自负皆来自我的自卑，所有的英雄气概都来自于我的软弱。

嘴里振振有词是因为心里满是怀疑，深情是因为痛恨自己无情。

这世界没有一件事情是虚空而生的，站在光里，背后就会有阴影。

这深夜里一片寂静，是因为你还没有听见声音。
```

#### 《一切》北岛

```
一切都是命运

一切都是烟云

一切都是没有结局的开始

一切都是稍纵即逝的追寻

一切欢乐都没有微笑

一切苦难都没有泪痕

一切语言都是重复

一切交往都是初逢

一切爱情都在心里

一切往事都在梦中

一切希望都带着注释

一切信仰都带着呻吟

一切爆发都有片刻的宁静

一切死亡都有冗长的回声
```
